## 简单介绍

一直都想在Github上做一个真正的项目，借着图像处理算法学习和毕设的契机，使用QT开发了一个用于测试和研究图像处理的UI界面。

目前包含的主要算法为基于截断核范数的图像恢复算法，后续会不断加入新的内容。欢迎大家讨论学习！

打开帮助以查阅帮助文档！
